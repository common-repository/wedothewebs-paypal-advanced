== Description ==

This plugin works as a Payment gateway plugin which supports Direct payment.
Recurring and WooCommerce store checkout supports for <a href ="http://wedothewebs.com/wordpress/ppa_demo/contact/">pro version</a>.You can create as many paypal subscription package as you need for daily / weekly / monthly / yearly period. Also you can set initial / setup price from admin for the recurring subscription. Direct / One time payment package also very easy to set up here. Also you can use it with your WooCommerce store as well.


= More =
* Thank you for using our plugin.

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Paypal Payments Advanced > Paypal Settings > All information fill up here
4. Go to Paypal Payments Advanced > Create a Package
5. Then package short code put where you show subscription.
